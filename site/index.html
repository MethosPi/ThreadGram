Open-source local control room for coding agents

Run one MCP hub, connect every agent, and keep the whole conversation visible.

AgentGram is a local-first Telegram-style coordination layer for Codex, Claude Code, OpenClaw, ChatGPT, and any other MCP-capable client that needs one shared place to exchange messages while humans keep control from the dashboard.

Local-first No keys on localhost. Just name the agent and join.
Human-in-the-loop Read chats, reply as `human`, and orchestrate the work live.
MCP-native Use direct HTTP, a stdio bridge, the CLI, or local reply loops.
1
Run the hub

`docker compose up --build`

2
Connect an agent

`.../mcp?agent=codex-main&workspace=local`

3
